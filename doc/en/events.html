Event

Method Description
on(event[, listener, capture]) Listen to the event event.
event
String, multiple event types are separated by spaces.
listener
Event When a specified type of event occurs, the object that receives notification (object that implements the interface).
capture
If set to true, it will execute during the capture process, otherwise it will execute the processing function during the bubbling process. If not specified, capture defaults to false.
off(event[, listener]) The function of the event handler removed from the event target.
event
String, multiple event types are separated by spaces.
listener
Same as listener in on.
bind(event[, callback]) Bind an event handler to the matched element, event is the same as above.
unbind(event) The unbound event of the matched element, and the event is the same as above.
blur(callback) Trigger or bind the function to the blur event of the specified element.
focus(callback) Trigger or bind the function to the focus event of the specified element.
focusin(callback) Trigger or bind the function to the focusin event of the specified element.
focusout(callback) Trigger or bind the function to the focusout event of the specified element.
resize(callback) Trigger or bind the function to the resize event of the specified element.
scroll(callback) Trigger or bind the function to the scroll event of the specified element.
click(callback) Trigger or bind the function to the click event of the specified element.
dblclick(callback) Trigger or bind the function to the double click event of the specified element.
mousedown(callback) Trigger or bind the function to the mouse down event of the specified element.
mouseup(callback) Trigger or bind the function to the mouse up event of the specified element.
mousemove(callback) Trigger or bind the function to the mouse move event of the specified element.
mouseover(callback) Trigger or bind the function to the mouse over event of the specified element.
mouseout(callback) Trigger or bind the function to the mouse out event of the specified element.
mouseenter(callback) Trigger or bind the function to the mouse enter event of the specified element.
mouseleave(callback) Trigger or bind the function to the mouse leave event of the specified element.
change(callback) Trigger or bind the function to the change event of the specified element.
selec(callback)t Trigger or bind the function to the select event of the specified element.
submit(callback) Trigger or bind the function to the submit event of the specified element.
keydown(callback) Trigger or bind the function to the key down event of the specified element.
keypress(callback) Trigger or bind the function to the key press event of the specified element.
keyup(callback) Trigger or bind the function to the key up event of the specified element.
contextmenu(callback) Trigger or bind the function to the contextmenu event of the specified element.
load(callback) Trigger or bind the function to the load event of the specified element.

Code

        // html
        <div>
            <button id="click">click</button>
            <button id="enter">mouse enter or leave</button>
            keyup: <input type="text" id="keyup">
        </div>

        // javascript
        function print(v) {
            qc("#cons").prepend("
" + v + "
");
        }

        qc("#click").click(function(ev) {
            print(ev.type);
        });

        qc("#enter").on("mouseenter mouseleave", function(ev) {
            print(ev.type);
        });

        qc("#keyup").bind("keyup", function(ev) {
            print(ev.code);
        });

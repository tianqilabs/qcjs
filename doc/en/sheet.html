sheet

Sheet view。

sheet attributes

qc-get
A link or method to get data.
parameter:
  • re: 对象。
  • callee: 回调函数。
qc-post
The link or method that is triggered when the selection changes.
qc-mode
manual: Load the data manually.
auto: The data is loaded automatically (default).
single: single choice (默认)
multiple: multiple choice
qc-key
Get the key field of the data, the unique value.
qc-text
JSON format, multi-select content corresponding options.

        function get1(re, callee) {
            var p = parseInt(re.args.page),
                count = parseInt(re.args.count),
                pages = Math.ceil(datas.length / count),
                start = (p - 1) * count,
                end = start + count;

            var data = datas.slice(start, end);
            var d = {rows: data.length, data: data, pages: pages, count: datas.length};
            callee(d, re);
        }

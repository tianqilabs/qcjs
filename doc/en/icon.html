Custom icon

The font icon of the qcjs widget uses Font Awesome by default.

To use other icons, please modify the qc.icon object, qc.util.setIconStyle method settings.

Description

qc.icon[widgetName] = {Name: Value}

widgetName:
Type: String
Widget name or object name

Use

Code

        // javascript
        qc.icon.popfrm = {
            "ok": "fa-check",
            "close": "fa-times"
            ...
        };

Methods

qc.util.setIconStyle(cont)

Description: Set the style of the icon.

cont
Type: htmlString or String
htmlString: The icon is set in html. String: The icon is set in the css of the Element before the single.

qc.util.icon(obj, name[, contrlName])

Description: Set the icon style of the current QC.

object
Type: QC
The current QC object.
name
Type: String
The attribute name of the object.
contrlName
Type: String
The name of the widget or object.
Default: the widget to which the object belongs.

Code

        // javascript
        qc.util.iconStyle("");
        qc.icon.popfrm = {
            "ok": "fa fa-check", // widget: popfrm, name: ok, class name: fa-check
            "close": "fa fa-times" // widget: popfrm, name: close, class name: fa-times
        };

        // html
        // OK button
        
        // Close button
         


        // javascript
        qc.util.iconStyle(" ");
        qc.icon.popfrm = {
            "ok": "fa-check",
            "close": "fa-times",
            "minus": "fa-minus"
        };

        // html
        // OK button
        
        // Close button
        
              

        // html
        
        

        // javascript
        qc.util.icon(qc("#icon1"), "popfrm", "ok");
        qc.util.icon(qc("#icon2"), "popfrm", "minus");

        // result html

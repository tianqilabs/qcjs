popfrm

Dialog box

Attributes

qc-for
Value: popfrm.
qc-target
A CSS selector that specifies the popfrm object.
qc-buttons
The popfrm object button, multiple separated by spaces.
Rules: [obj1, obj2...] | ["ok", "cancel"] | [obj1, "ok"].
obj1-objN: {name: name, title: title, icon: icon class name, text: displayed text content, fn: click to trigger the event}.
ok|cance: string, cancel will trigger the qc.popfrm.hide method, ok currently only defines the icon, there is no execution method, you need to specify fn.
qc-title
The title of popfrm.
qc-content
The content of popfrm.

Dynamic call-javascript

Code

        // html body
        <p>
            <button id="pop6">dyShow popfrm</button>
        </p>
        <p>
            <button id="pop7">dyShow popfrm and OK</button>
        </p>
        <p>
            <button id="pop8">dyShow popfrm, not toolbar</button>
        </p>

        // javascript
        function pop2_OK(obj) {
            alert("button title: " + obj.curr.attr("title"));
            qc.popfrm.hide(obj);
        }

        qc(function () {
            qc("#pop6").click(function () {
                qc.popfrm.dyShow("Tips", [], "This is a popfrm.", qc(this));
            });

            qc("#pop7").click(function () {
            qc.popfrm.dyShow("Tips", [
                {"name": "ok", "fn": "pop2_OK", "title": "pop4-OK"},
                {"name": "minus", "fn": "pop2_OK", "title": "pop4-minus", "icon": "fa-minus", "text": "minus"}
                ],
            });

            qc("#pop8").click(function () {
                qc.popfrm.dyShow("Tips", "",
                    "This is a popfrm.", qc(this), null, function() {
                    alert("hidden ");
                });
            });

        });

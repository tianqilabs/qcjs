util

qc.util.init([...obj])

obj:
Type: QC Widget
One or more widget controls

qc.util.addStart(fn)

fn:
Type: Function
util The method to be executed before initializing the widget.

qc.util.addInited(fn)

fn:
Type: Function
util The method to be executed after initializing the widget.

qc.util.getKeys(contrl[, args])

Description: Generate an args object based on the qc-key and qc-need of the widget. And generate the key attribute and value.

contrl:
Type: QC Widget
QC widget.
args:
Type: Object
Submit the data object.
return:
Type: Object
Return the object.

qc.util.getFields(contrl[, attr, args])

Description: Generate an args object based on the attr attribute of the widget. And generate the key attribute and value.

contrl:
Type: QC Widget
QC widget.
attr:
Type: String
Generally refers to qc-key, qc-need, etc. The attr parameter is not defined, and all the qc-fields of the widget are traversed.
return:
Type: Object
Return the object.

qc.util.getVal(obj[, args])

Description: Update the qc-value attribute of the object and return the qc-value. If you define args, set the args object.

obj:
Type: QC Object
QC object.
args:
Type: Object
Submit the data object.
return:
Type: String or Array or HTMLInputElement
Returns the qc-value value.

qc.util.get(obj[, args, callback])
qc.util.post(obj[, args, callback])
qc.util.upload(obj[, args, callback])

Description: get: get data, post: submit data, upload: upload file. Corresponding widget attributes qc-get, qc-post, qc-upload.

obj:
Type: QC Object
QC object.
args:
Type: Object
Submit the data object.
callback:
Type: Function(d, re)

Callback method.

d:

Type: JSON Object

{"rows": array length, "data": array, "pages": total number of pages, "total": total number},

re:

Type: Object

{"contrl": widget object, "curr": current object, "args": parameters when transferring data}.

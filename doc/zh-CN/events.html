事件

方法 描述
on(event[, listener, capture]) 监听 event 事件。
event
字符串，多个事件类型用空格分开。
listener
Event当指定类型的事件发生时，接收通知的对象 (实现接口的对象)。
capture
设置为 true 就在捕捉过程中执行,反之就在冒泡过程中执行处理函数。如果未指定,则 capture 默认为 false。
off(event[, listener]) 从事件目标中删除的事件处理程序的功能。
event
字符串，多个事件类型用空格分开。
listener
与 on 中的 listener 相同。
bind(event[, callback]) 向匹配元素绑定事件处理器，event 同上。
unbind(event) 匹配元素解除绑定事件，event 同上。
blur(callback) 触发、或将函数绑定到指定元素的 blur 事件。
focus(callback) 触发、或将函数绑定到指定元素的 focus 事件。
focusin(callback) 触发、或将函数绑定到指定元素的 focusin 事件。
focusout(callback) 触发、或将函数绑定到指定元素的 focusout 事件。
resize(callback) 触发、或将函数绑定到指定元素的 resize 事件。
scroll(callback) 触发、或将函数绑定到指定元素的 scroll 事件。
click(callback) 触发、或将函数绑定到指定元素的 click 事件。
dblclick(callback) 触发、或将函数绑定到指定元素的 double click 事件。
mousedown(callback) 触发、或将函数绑定到指定元素的 mouse down 事件。
mouseup(callback) 触发、或将函数绑定到指定元素的 mouse up 事件。
mousemove(callback) 触发、或将函数绑定到指定元素的 mouse move 事件。
mouseover(callback) 触发、或将函数绑定到指定元素的 mouse over 事件。
mouseout(callback) 触发、或将函数绑定到指定元素的 mouse out 事件。
mouseenter(callback) 触发、或将函数绑定到指定元素的 mouse enter 事件。
mouseleave(callback) 触发、或将函数绑定到指定元素的 mouse leave 事件。
change(callback) 触发、或将函数绑定到指定元素的 change 事件。
selec(callback)t 触发、或将函数绑定到指定元素的 select 事件。
submit(callback) 触发、或将函数绑定到指定元素的 submit 事件。
keydown(callback) 触发、或将函数绑定到指定元素的 key down 事件。
keypress(callback) 触发、或将函数绑定到指定元素的 key press 事件。
keyup(callback) 触发、或将函数绑定到指定元素的 key up 事件。
contextmenu(callback) 触发、或将函数绑定到指定元素的 contextmenu 事件。
load(callback) 触发、或将函数绑定到指定元素的 load 事件。

Code

        // html
        <div>
            <button id="click">click</button>
            <button id="enter">mouse enter or leave</button>
            keyup: <input type="text" id="keyup">
        </div>

        // javascript
        function print(v) {
            qc("#cons").prepend("
" + v + "
");
        }

        qc("#click").click(function(ev) {
            print(ev.type);
        });

        qc("#enter").on("mouseenter mouseleave", function(ev) {
            print(ev.type);
        });

        qc("#keyup").bind("keyup", function(ev) {
            print(ev.code);
        });

datepicker

时间选择器。

属性

qc-for
值：datepicker。
qc-target
返回日期值的对象或元素，CSS selector 字符串。
qc-mode
date: 默认，日期方式。
datetime: 日期时间方式。
time: 时间方式。

Code

        // html body
        qc-mode: date (default)
        <input type="text" qc-for="datepicker" class="datepicker" value="2020-2-3">

        qc-mode: datetime
        <input type="text" qc-for="datepicker" qc-mode="datetime" class="datepicker">

        qc-mode: time
        <input type="text" qc-for="datepicker" qc-mode="time"  class="datepicker">

Result

qc-mode: date (default)


qc-mode: datetime


qc-mode: time

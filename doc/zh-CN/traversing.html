方法 描述
first(selector) 将匹配元素集合缩减为集合中的第一个元素。
last 将匹配元素集合缩减为集合中的最后一个元素。
move(dir[, selector, once]) 获得匹配元素集合中每个元素紧邻的同辈元素。
dir
移动方向，next (后面) 或 prev (前面) 或 parent (父级)，默认: next。
selector
可选，如果提供选择器，则根据 dir 的方向取回匹配该选择器的一个同胞元素。
once
可选，boolean 类型，默认 false。是否只移动一次。
prev(selector) 获得匹配元素集合中每个元素紧邻的同胞前一个元素。如果提供选择器，匹配是否相同。
prevAll(selector) 获得匹配元素集合中每个元素紧邻的同胞前所有元素。如果提供选择器，则取匹配相同。
next(selector) 获得匹配元素集合中每个元素紧邻的同胞后一个元素。如果提供选择器，匹配是否相同。
nextAll(selector) 获得匹配元素集合中每个元素紧邻的同胞后所有元素。如果提供选择器，则取匹配相同。
parent(selector) 获得匹配元素集合中每个元素的一个父级元素。如果提供选择器，匹配是否相同。
parents(selector) 获得匹配元素集合中每个元素的所有父级元素。如果提供选择器，则取匹配相同。
closest(selector) 方法获得匹配选择器的第一个祖先元素，从当前元素开始沿 DOM 树向上。
filter(selector) 将匹配元素集合缩减为匹配指定选择器的元素。
contents(selector) 获得匹配元素集合中每个元素的子节点，包括文本和注释节点。
eq(index) 匹配元素集缩减值指定 index 上的一个。
find(selector) 获得当前元素集合中每个元素的后代，通过选择器、QC 对象或元素来筛选。
is(selector) 根据选择器、元素或 QC 对象来检测匹配元素集合，如果这些元素中至少有一个元素匹配给定的参数，则返回 true。
index() 获得第一个匹配元素相对于父节点的子节点的 index 位置， 不包含文本和注释节点。
indexAll(isEl) 获得第一个匹配元素相对于父节点的子节点的 index 位置， 包含文本和注释节点。
isEl
可选。

isEl == true 时，返回同级节点为 Element 的 index 位置，等同 index() 方法。
isEl == undefined || false 时，返回同级节点的 index 位置 (包含文本和注释节点)。

Code

        // html
        <ul>
            <li>a1</li>
            <li>a2</li>
            <li class="a3">a3</li>
            <li>a4</li>
            <li>a5</li>
        </ul>

        // javascript
        function print(obj) {
            var ary = [];
            obj.each(function () {
                ary.push(this.outerHTML);
            });
            console.log(ary.join("\n"));
        }

        var lis = qc("li");

        print(lis.first());     // <li>a1</li>
        print(lis.last());      // <li>a5</li>

        var a3 = lis.eq(2);
        print(a3);              // <li class="a3">a3</li>

        print(a3.prev());       // <li>a2</li>

        print(a3.prevAll());    // <li>a2</li>
                                // <li>a1</li>

        print(a3.next());       // <li>a4</li>

        print(a3.nextAll());    // <li>a4</li>
                                // <li>a5</li>

        print(a3.closest("ul"));       // <ul>
                                       //     <li>a1</li>
                                       //     <li>a2</li>
                                       //     <li class="a3">a3</li>
                                       //     <li>a4</li>
                                       //     <li>a5</li>
                                       // </ul>

        print(a3.parent());
        print(a3.parents("ul"));       // <ul>
                                       //     <li>a1</li>
                                       //     <li>a2</li>
                                       //     <li class="a3">a3</li>
                                       //     <li>a4</li>
                                       //     <li>a5</li>
                                       // </ul>

        print(lis.filter(".a3"));      // <li class="a3">a3</li>

        print(qc("ul").contents());    // <li>a1</li>
                                       // <li>a2</li>
                                       // <li class="a3">a3</li>
                                       // <li>a4</li>
                                       // <li>a5</li>

        var a31 = lis.find(".a3");
        print(a31);                    // <li class="a3">a3</li>

        console.log(a31.is(".a3"));    // true

        console.log(a31.index());      // 2

        console.log(a31.indexAll());   // 5

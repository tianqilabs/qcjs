自定义图标

qcjs 小部件的字体图标默认使用 Font Awesome。

要使用其它图标，请修改 qc.icon 对象，qc.util.setIconStyle 方法设置。

说明

qc.icon[widgetName] = {名称: 值}

widgetName:
Type：String
小部件名称或对象名

使用

Code

        // javascript
        qc.icon.popfrm = {
            "ok": "fa-check",
            "close": "fa-times"
            ...
        };

方法

qc.util.setIconStyle(cont)

描述：设置图标使用样式。

cont
Type: htmlString or String
htmlString: 图标设置在 html 内。 String: 图标设置在单前 Element 的 css。

qc.util.icon(obj, name[, contrlName])

描述：设置当前 QC 的图标样式。

object
Type: QC
当前 QC 对象。
name
Type: String
对象的属性名称。
contrlName
Type: String
小部件或对象名称。
默认：对象所属的小部件。

Code

        // javascript
        qc.util.iconStyle("");
        qc.icon.popfrm = {
            "ok": "fa fa-check",     // 小部件: popfrm，名称: ok，class 名称: fa-check
            "close": "fa fa-times"   // 小部件: popfrm，名称: close，class 名称: fa-times
        };

        // html
        // OK 按钮
        <button class="fa fa-check"> OK </button>
        // Close 按钮
        <a href="#" class="fa fa-times"> </a>


        // javascript
        qc.util.iconStyle(" ");
        qc.icon.popfrm = {
            "ok": "fa-check",
            "close": "fa-times",
            "minus": "fa-minus"
        };

        // html
        // OK 按钮
        <button>
            <i class="fa fa-check"> </i> Check
        </button>
        // Close 按钮
        <a href="#">
            <i class="fa fa-times"> </i>
        </a>

        // html
        <button id="icon1"> OK </button>
        <button id="icon2"> minus </button>

        // javascript
        qc.util.icon(qc("#icon1"), "popfrm", "ok");
        qc.util.icon(qc("#icon2"), "popfrm", "minus");

        // result html

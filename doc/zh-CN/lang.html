多国语言

qcjs 小部件使用到的按钮、标题支持多国语言。

文档在 lang 文件夹里，命名方式：en.js (英文)，zh-CN.js (简体中文)。

说明

qc.lang[widgetName] = {名称: 值}

widgetName:
Type：String
小部件名称或对象名

Code

        // global.js
        qc.lang.global = {
            "en": "English",
            "zh-CN": "中文（简体）"
        };

        // zh-CN.js
        qc.lang.popfrm = {
            "OK": "确定",         // 小部件: popfrm，名称: OK，值: 确定
            "cancel": "取消",      // 小部件: popfrm，名称: cancel，值: 取消
            ...
        };

        // en.js
        qc.lang.popfrm = {
            "OK": "OK",
            "cancel": "Cancel",
            ...
        };

使用

Code

        // html 标签
        <html lang="zh-CN" qc-lang-dir="../lang" qc-lang-global="global">

        // javascript:
        qc.util.setLangDir('../lang');
        qc.util.setLangGlobal('global');
        qc.util.setLangName('zh-CN');

方法

qc.util.setLangDir(dirPath)

dirPath
Type: String
多国语言路径

qc.util.getLangDir()

return
Type: String
返回多国语言路径

qc.util.setLangGlobal(fileName)

fileName
Type：String
设置语言列表文件名。

qc.util.getLangGlobal()

return
Type：String
返回语言列表文件名。

qc.util.setLangName(langName)

fileName:
Type：String
设置当前使用的语言。

qc.util.getLangName()

return
Type：String
返回当前使用的语言。

qc.util.initLang([langName, callback])

初始化语言

langName
Type：String
当前使用的语言。
callback
Type：Function
回调方法。

qc.util.lang(obj, name[, attr, controlName])

设置语言

obj
Type：QC
要设置的 QC 对象。
name
Type：String
对象的属性名称。
attr
Type：String
Html 元素属性，html: 替换 Html 元素的内容，title/value: 替换 Html 元素的属性。
支持多个属性，英文逗号分隔符。
默认：title
controlName
Type：String
小部件或对象名称。
默认：对象所属的小部件。

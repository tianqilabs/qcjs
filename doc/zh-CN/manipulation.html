操作

insert(cont[, type])

描述：将 cont 按 type 类型插入到集合中的每个元素。

cont：
Type：selector or Element or QC or Node or NodeList or elementArray or htmlString
选择器、DOM 元素、节点、节点集、QC、html 字符串。
type：
Type：String
append (默认)、prepend、before、after。
return：
QC 对象

append(cont)

描述：将 cont 插入到集合中的每个元素的末尾。

cont：
与 insert 的 cont 相同。

prepend(cont)

描述：将 cont 插入到集合中的每个元素的开头。

cont：
与 insert 的 cont 相同。

before(cont)

描述：将 cont 插入到集合中的每个元素的前面。

cont：
与 insert 的 cont 相同。

after(cont)

描述：将 cont 插入到集合中的每个元素的后面。

cont：
与 insert 的 cont 相同。

remove()

描述：从DOM中删除匹配的元素集。

clone()

描述：创建匹配元素集的副本。

empty()

描述：从列表中删除所有。

attr(name[, value])

描述：设置或返回属性。

name：
属性名称。
value：
属性值。

removeAttr(name)

描述：删除指定名称的属性。

name：
属性名称。

html(str)

描述：设置或返回第一个元素的 innerHTML。

str:
Html 字符串。

text(str)

描述：设置或返回第一个元素的 Text。

str:
字符串。

css(name[, value])

描述：设置或返回元素的 style。

name:
CSS 名称。
value:
CSS 值。

popfrm

对话框

属性

qc-for
值：popfrm。
qc-target
一个 CSS selector，指定 popfrm 对象。
qc-buttons
popfrm 对象按钮，多个用空格分隔。
规则：[obj1, obj2...] | ["ok", "cancel"] | [obj1, "ok"]。
obj1 - objN: {name: 名称, title: 标题, icon: 图标类名, text: 显示的文本内容, fn: 点击触发的事件}。
ok|cance: string，cancel 会触发 qc.popfrm.hide 方法，ok 目前只定义了图标，没有执行方法，需要指定 fn。
qc-title
popfrm 的标题。
qc-content
popfrm 的内容。

动态调用 - javascript

Code

        // html body
        <p>
            <button id="pop6">dyShow popfrm</button>
        </p>
        <p>
            <button id="pop7">dyShow popfrm and OK</button>
        </p>
        <p>
            <button id="pop8">dyShow popfrm, not toolbar</button>
        </p>

        // javascript
        function pop2_OK(obj) {
            alert("button title: " + obj.curr.attr("title"));
            qc.popfrm.hide(obj);
        }

        qc(function () {
            qc("#pop6").click(function () {
                qc.popfrm.dyShow("Tips", [], "This is a popfrm.", qc(this));
            });

            qc("#pop7").click(function () {
            qc.popfrm.dyShow("Tips", [
                {"name": "ok", "fn": "pop2_OK", "title": "pop4-OK"},
                {"name": "minus", "fn": "pop2_OK", "title": "pop4-minus", "icon": "fa-minus", "text": "minus"}
                ],
            });

            qc("#pop8").click(function () {
                qc.popfrm.dyShow("Tips", "",
                    "This is a popfrm.", qc(this));
            });

        });

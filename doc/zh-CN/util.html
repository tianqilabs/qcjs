util

qc.util.init([...obj])

obj:
Type：QC Widget
一个或多个小部件控件

qc.util.addStart(fn)

fn:
Type：Function
util 初始化小部件前执行的方法。

qc.util.addInited(fn)

fn:
Type：Function
util 初始化小部件结束后执行的方法。

qc.util.getKeys(contrl[, args])

描述：根据小部件的 qc-key、qc-need 生成 args 对象。并生成 key 属性和值。

contrl:
Type：QC Widget
QC 小部件。
args:
Type：Object
提交数据对象。
return:
Type：Object
返回对象。

qc.util.getFields(contrl[, attr, args])

描述：根据小部件的 attr 属性生成 args 对象。并生成 key 属性和值。

contrl:
Type：QC Widget
QC 小部件。
attr:
Type：String
一般指 qc-key、qc-need 等。没有定义 attr 参数，历遍小部件所有 qc-field。
return:
Type：Object
返回对象。

qc.util.getVal(obj[, args])

描述：更新对象的 qc-value 属性，返回 qc-value。如果定义 args，设置 args 对象。

obj:
Type：QC Object
QC 对象。
args:
Type：Object
提交数据对象。
return:
Type：String or Array or HTMLInputElement
返回 qc-value 值。

qc.util.get(obj[, args, callback])
qc.util.post(obj[, args, callback])
qc.util.upload(obj[, args, callback])

描述： get: 获取数据，post: 提交数据，upload: 上传文件。 相对应小部件属性 qc-get、qc-post、qc-upload。

obj:
Type：QC Object
QC 对象。
args:
Type：Object
提交数据对象。
callback:
Type：Function(d, re)

回调方法。

d:

Type：JSON Object

{"rows": 数组长度, "data": 数组, "pages": 总页数, "total": 总数量}，

re:

Type：Object

{"contrl": 小部件对象, "curr": 当前对象, "args": 传送数据时的参数}。

Html 属性

名称 描述
qc-control 指定小部件名称
qc-type 类型。
如果定义了 qc-fn，检验 qc-type 值与事件类型，true 时执行 qc-fn 方法。
没定义，执行 qc-type 方法。多个用空格分隔。
qc-page 当前页码，与 qc-count 配对使用，默认 1。
qc-count 每页数量，与 qc-page 配对使用，默认 20。
qc-fn 方法。
如果指定 qc-type，检验 qc-type 值与事件类型，true 时执行 qc-fn 方法。
没定义，执行 qc-fn 方法。
qc-mode 小部件模式，请参照各个小部件说明。
qc-field 数据字段名称。
qc-default 数据字段缺省值。
qc-value 数据字段值。
qc-key 提交数据的关键字段。在 qc-get 或 qc-post 时会生成 key: qc-key 名称 ，多个用英文逗号分隔。
qc-need 提交数据的必需同时提交的字段，多个用英文逗号分隔。
qc-subfield 子字段属性。
qc-text 扩展设置属性。
qc-target 一个 CSS selector，指定目标小部件或 Html 元素。
qc-get 获取数据的路径或方法。
qc-post 提交数据的路径或方法。
qc-upload 上传文件的路径或方法。
qc-for 一个 CSS selector，绑定小部件。
popfrm、colorpicker，selector 触发 show 方法。
qc-rule 小部件规则。
qc-warp 背景属性。
qc-container 容器属性。
qc-content 内容属性。
qc-toolbar 小部件工具栏。
qc-title 小部件标题。
qc-buttons 小部件按钮组。
qc-success 验证成功，一个 CSS selector。
qc-failure 验证失败，一个 CSS selector。
qc-lang 小部件或对象名称。
qc-lang-name 对象的属性名称。
qc-lang-attr 设置 qc.lang 对象值的方式，html: 替换 Html 元素的内容，title/value: 替换 Html 元素的属性。
qc-colorpicker-mode qc-colorpicker-mode 属性。
设置 qc-colorpicker 的模式：list (颜色列表)，panel (面板选择器)。
qc-colorpicker-css qc-colorpicker-css 属性。
要设置和获取的 css 名称。

        // html
        <p>
            <button id="pop1" qc-for="popfrm" qc-target="#pop-frm1">
                Show popfrm, not toolbar</button>
        </p>
        <div class="qc-popfrm" qc-control="popfrm" id="pop-frm1">
            <div class="qc-container" qc-container>
                <div class="qc-toolbar" qc-toolbar>
                    <span  class="qc-title" qc-title>Tips</span>
                </div>
                <div class="qc-content" qc-content>
                    This is a popfrm.
                </div>
            </div>
        </div>

        <p>
            <button id="pop2" qc-for="popfrm" qc-target="#pop-frm2">
                Show popfrm toolbar</button>
        </p>
        <div class="qc-popfrm" qc-control="popfrm" id="pop-frm2">
            <div class="qc-container" qc-container>
                <div class="qc-toolbar" qc-toolbar>
                    <span  class="qc-title" qc-title>Tips</span>
                    <div class='qc-buttons' qc-buttons>
                        <a  href='javascript:void(0);' qc-fn="pop2_OK" title="OK">
                            <i class="fa fa-check"> </i>
                        </a>
                        <a  href='javascript:void(0);' qc-title="Close" qc-type="hide">
                            <i class="fa fa-times"> </i>
                        </a>
                    </div>
                </div>
                <div class="qc-content" qc-content>
                    This is a popfrm.
                </div>
            </div>
        </div>

        <p>
            <button id="pop3">dyShow popfrm</button>
        </p>

        // javascript
        function pop2_OK(obj) {
            alert("button title: " + obj.curr.attr("title"));
            qc.popfrm.hide(obj);
        }

        qc(function () {
            qc("#pop3").click(function () {
                qc.popfrm.dyShow("Tips", [], "This is a popfrm.", qc(this));
            });

        });

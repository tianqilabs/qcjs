Multi-language

The buttons and titles used by the qcjs widget support multiple languages.

The document is in the lang folder, naming method: en.js (English), zh-CN.js (Simplified Chinese).

Description

qc.lang[widgetName] = {Name: Value}

widgetName:
Type: String
Widget name or object name

Code

        // global.js
        qc.lang.global = {
            "en": "English",
            "zh-CN": "Chinese (Simplified)"
        };

        // zh-CN.js
        qc.lang.popfrm = {
            "OK": "OK", // widget: popfrm, name: OK, value: OK
            "cancel": "Cancel", // widget: popfrm, name: cancel, value: cancel
            ...
        };

        // en.js
        qc.lang.popfrm = {
            "OK": "OK",
            "cancel": "Cancel",
            ...
        };

Use

Code

        // html tag
        

        // javascript:
        qc.util.setLangDir('../lang');
        qc.util.setLangGlobal('global');
        qc.util.setLangName('zh-CN');

Methods

qc.util.setLangDir(dirPath)

dirPath
Type: String
Multi-language path

qc.util.getLangDir()

return
Type: String
Return to multi-language path

qc.util.setLangGlobal(fileName)

fileName
Type: String
Set the language list file name.

qc.util.getLangGlobal()

return
Type: String
Returns the file name of the language list.

qc.util.setLangName(langName)

fileName:
Type: String
Set the current language.

qc.util.getLangName()

return
Type: String
Returns the language currently in use.

qc.util.initLang([langName, callback])

Initialization language

langName
Type: String
The language currently used.
callback
Type: Function
Callback method.

qc.util.lang(obj, name[, attr, controlName])

Set language

obj
Type: QC
The QC object to be set.
name
Type: String
The attribute name of the object.
attr
Type: String
Html element attributes, html: replace the content of the Html element, title/value: replace the attributes of the Html element.
Support multiple attributes, English comma separator.
Default: title
controlName
Type: String
The name of the widget or object.
Default: the widget to which the object belongs.

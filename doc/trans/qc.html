QC object

Description: Accept a string containing CSS selectors, and then use it to match a set of elements.

qc(selector[, context])

selector:
Type: Selector
Must be a legal CSS selector string
context:
Type: Element or QC
DOM element, document, QC used as the context, the current page document is the default.
return:
QC object

qc(object)

object:
Type: Element or Node or NodeList or ElementArray
Element element, Node node, NodeList set, Element array
return:
QC object

qc(html[, document])

html:
Type: htmlString
A dynamically created HTML string.
document:
Type: Document
The document in which the new element will be created.
return:
QC object

Attributes

length
The number of elements in QC.
Element access
For similar numerical operations, use [index], the index where the index element is located. Return the Element object.

Methods

each(callback)

Description: Traverse the elements in QC.

callback:
Type: Function(index, thisObject)
The function to be executed on each value, this object is the Element element corresponding to QC.

index: The index in QC.

thisObject: QC.

Tips

return true Return, continue the loop. Similar to continue.

return false Return to stop the loop. Similar to break.

Static method

qc._indexOf(obj, search[, index])
Description: search is at the position of obj, the position where index starts to search.
qc._extend(obj[, ...objN])
Merge the contents of two or more objects into the first object.
qc._each(obj, callback)
Traverse the obj object.
callback:

Type: Function(value, index, thisObject)

value: The current object.

index: current position.

thisObject: obj object.

qc._equals(obj1[, obj2])
Compare two objects and return boolean.
qc._convert(cont)
Convert the string cont to Text Node and return the QC object.
qc._matches(el, str)
Verify that the specified str and el match.
el: Element element.
str: The string to be verified, legal CSS selector.
qc._ajax(url[, settings])
Perform asynchronous HTTP (Ajax) requests.
url: A string containing the URL to which the request was sent.
settings:
type:

Send type, currently only supports get and post.

str:

The string to be verified, legal CSS selector.

data:

The submitted parameters.

For example: {"name": "Tom", "age": 32, "gender": "man"}.

// Name: Tom, age: 32 years old, gender: male.

Type of data value: String, Number, Array, . progress:

Only for uploaded files, upload progress event (IE9 not supported).

callback:

event.loaded The amount of data currently transferred.
event.total The total amount of data to be transferred.

error:

Only for uploaded files, upload error events (IE9 not supported).

success:

Submit a successful event.

callback:

The content returned by the server.

qc._post(url, data[, callback])

Send data in post mode.

url: A string containing the URL to which the request was sent.
data: The submitted parameter object.
callback: callback method.
qc._get(url, data[, callback])

Use get to send data.

url: A string containing the URL to which the request was sent.
data: The submitted parameter object.
callback: callback method.
qc.getScript(url, [doc, callback])

Load JavaScript files dynamically.

url: A string containing the URL to which the request was sent.
doc: The document object loaded into.
callback: callback method.

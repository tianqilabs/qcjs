Name Description
qc-control Specify the widget name
qc-type Type.
If qc-fn is defined, check the value of qc-type and event type. If true, execute the qc-fn method.
If not defined, execute the qc-type method. Multiples are separated by spaces.
qc-page The current page number, paired with qc-count, the default is 1.
qc-count The number of pages per page, paired with qc-page, the default is 20.
qc-fn method.
If qc-type is specified, check the value of qc-type and event type. If true, execute the qc-fn method.
If not defined, execute the qc-fn method.
qc-mode Widget mode, please refer to the description of each widget.
qc-field The name of the data field.
qc-default The default value of the data field.
qc-value Data field value.
qc-key The key field of the submitted data. During qc-get or qc-post, key: qc-key name will be generated, and multiples are separated by spaces.
qc-need Field that is required to submit data concurrently filed.
qc-subfield Subfield attributes.
qc-text Extended setting attributes.
qc-target A CSS selector that specifies the target widget or Html element.
qc-get The path or method to obtain the data.
qc-post The path or method of submitting data.
qc-upload The path or method of the uploaded file.
qc-for A CSS selector to bind widgets.
popfrm, colorpicker, and selector trigger the show method.
qc-rule Rules for widgets.
qc-warp background properties.
qc-container Container attributes.
qc-content Content attributes.
qc-toolbar Widget toolbar.
qc-title Widget title.
qc-buttons Widget button group.
qc-success The verification is successful, a CSS selector.
qc-failure Validation failed, a CSS selector.
qc-lang The name of the widget or object.
qc-lang-name The attribute name of the object.
qc-lang-attr The way to set the value of the qc.lang object, html: replace the content of the Html element, title/value: replace the attribute of the Html element.
qc-colorpicker-mode qc-colorpicker-mode properties.
Set the mode of qc-colorpicker: list (color list), panel (panel selector).
qc-colorpicker-css qc-colorpicker-css attribute.
The css name to be set and obtained.

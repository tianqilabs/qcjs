Operation

insert(cont[, type])

Description: Insert cont into each element in the collection according to type.

cont:
Type: selector or Element or QC or Node or NodeList or elementArray or htmlString
Selector, DOM element, node, node set, QC, html string.
type:
Type: String
append (default), prepend, before, after.
return:
QC object

append(cont)

Description: Insert cont at the end of each element in the collection.

cont:
is the same as cont of insert.

prepend(cont)

Description: Insert cont at the beginning of each element in the collection.

cont:
is the same as cont of insert.

before(cont)

Description: Insert cont in front of each element in the collection.

cont:
is the same as cont of insert.

after(cont)

Description: Insert cont after each element in the collection.

cont:
is the same as cont of insert.

remove()

Description: Remove the set of matched elements from the DOM.

clone()

Description: Create a copy of the set of matching elements.

empty()

Description: Delete all from the list.

attr(name[, value])

Description: Set or return attributes.

name:
The name of the attribute.
value:
Attribute value.

removeAttr(name)

Description: Delete the attribute with the specified name.

name:
The name of the attribute.

html(str)

Description: Set or return the innerHTML of the first element.

str:
Html string.

text(str)

Description: Set or return the Text of the first element.

str:
string.

css(name[, value])

Description: Set or return the style of the element.

name:
CSS name.
value:
CSS value.

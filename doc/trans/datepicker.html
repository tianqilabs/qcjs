datepicker

Time picker.

Attributes

qc-for
Value: datepicker.
qc-target
The object or element that returns the date value, CSS selector string.
qc-mode
date: Default, date mode.
datetime: Date and time mode.
time: Time method.

Code

        // html body
        qc-mode: date (default)
        <input type="text" qc-for="datepicker" class="datepicker" value="2020-2-3">

        qc-mode: datetime
        <input type="text" qc-for="datepicker" qc-mode="datetime" class="datepicker">

        qc-mode: time
        <input type="text" qc-for="datepicker" qc-mode="time"  class="datepicker">

Result

qc-mode: date (default)


qc-mode: datetime


qc-mode: time
